# 《RiskCat》AE 演示视频脚本（仿苹果发布会风格，30–45s）

> 参考节奏：强节奏、黑底留白、轻发光、镜头丝滑推进
> 推荐时长：36s（6镜头 × 6s）

## S01（0:00–0:06）开场
- 画面：黑底 + 网格 + 雷达扫描 + 品牌字
- 文案：`RiskCat`
- 副文案：`多模态反诈助手`
- 动效：雷达扇形旋转 + 波纹扩散 + 标题淡入上移

## S02（0:06–0:12）输入
- 画面：三个输入入口图标（文本/截图/语音）
- 文案：`三路输入 即刻识别`
- 动效：图标依次点亮（间隔8帧）+ 光圈脉冲

## S03（0:12–0:18）识别
- 画面：UI 面板被扫描覆盖，证据点依次出现
- 文案：`证据链自动生成`
- 证据点：
  - `异常号码归属`
  - `诱导转账话术`
  - `高仿链接风险`
- 动效：扫描掠过触发标签弹出 + HUD描边

## S04（0:18–0:24）预警
- 画面：风险分数 + 红色高危卡片
- 文案：`风险上升 立即预警`
- 动效：
  - 分数 22→87（Slider 控制）
  - 卡片 Scale 80→105→100，Opacity 0→100
  - 红色扫描线竖向扫过

## S05（0:24–0:30）联动
- 画面：监护人通知卡片、侧边设备面板
- 文案：`联动守护 实时触达`
- 动效：通知卡右侧滑入 + 微回弹

## S06（0:30–0:36）结果
- 画面：报告生成页（摘要+建议）+ Logo 收尾
- 文案：`报告生成 闭环处置`
- Slogan：`RiskCat · 让诈骗无处遁形`
- 动效：模块分层上浮 + 全局 push-in + 收束停留

---

## AE 必做表达式（S04 分数）
```js
Math.round(thisComp.layer("Null 1").effect("Score")("Slider"))
```

## 全局统一参数
- 合成：1920×1080 / 30fps
- Motion Blur：开启
- 调色：Glow（轻）+ Noise 2~4% + Curves 压黑提亮
- 转场：Push-in / Match Cut / Light Sweep（克制）
