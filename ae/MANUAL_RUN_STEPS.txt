A. 在 After Effects 里执行脚本
1) 打开 After Effects 2026。
2) File > Scripts > Run Script File... 选择：C:\Users\FIRST\Desktop\projects\ae\build_project.jsx
3) 确认工程已生成并保存为：C:\Users\FIRST\Desktop\projects\ae\AntiFraud_Radar.aep
4) 再次 File > Scripts > Run Script File... 选择：C:\Users\FIRST\Desktop\projects\ae\render_queue.jsx
5) Render Queue 点击 Render，输出：C:\Users\FIRST\Desktop\projects\ae\out\main.mov

B. 导出 H.264 到 final.mp4
自动方案（Media Encoder）：
1) AE 菜单：Composition > Add to Adobe Media Encoder Queue
2) 格式选 H.264
3) 输出路径设为：C:\Users\FIRST\Desktop\projects\ae\out\final.mp4
4) 在 AME 点击 Start Queue

手动方案（ffmpeg）：
ffmpeg -y -i C:\Users\FIRST\Desktop\projects\ae\out\main.mov -c:v libx264 -pix_fmt yuv420p -preset medium -crf 18 -c:a aac -b:a 320k C:\Users\FIRST\Desktop\projects\ae\out\final.mp4
